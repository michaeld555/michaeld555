## Sobre mim

Atuo em todo o ciclo de vida do software, do levantamento de requisitos à entrega e manutenção. Minha principal especialidade é o ecossistema **PHP e Laravel**, aliando backend, interfaces web e desenvolvimento mobile para construir soluções completas e escaláveis.

- **Produtos:** ERPs financeiros, sistemas de gestão, e-commerces e plataformas digitais
- **Integrações:** APIs, pagamentos e serviços bancários com Mercado Pago, Sicoob e Santander
- **Infraestrutura e entrega:** Docker, CI/CD, GitHub, GitLab, AWS e Azure
- **Práticas:** código limpo, boas práticas, Scrum e Kanban
- **Pronomes:** ele/dele
